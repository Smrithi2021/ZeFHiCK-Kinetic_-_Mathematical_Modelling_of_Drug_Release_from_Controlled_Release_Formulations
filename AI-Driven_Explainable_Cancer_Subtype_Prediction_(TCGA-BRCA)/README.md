# AI-Driven Explainable Cancer Subtype Prediction (TCGA-BRCA)

# Repository Structure - `data/` → placeholder for raw and processed datasets (raw data not included)


Raw TCGA-BRCA datasets are **not included** due to size and access restrictions. You can obtain them from the [Data Portal](https://xenabrowser.net/datapages/) and place them in `data/raw/`.

1. Under "Public Datapages", scroll or search for "TCGA Breast Cancer (BRCA)".

2. Click on "TCGA Breast Cancer (BRCA)" to open its dataset page.

"TCGA Breast Cancer (BRCA) - RNAseq (gene expression)"

"TCGA BRCA – Phenotype (clinical patient/sample info)"

Download them as .tsv files

3. Place both downloaded files inside data/raw/
